# Typography

Our typography system is based on Bootstrap 5 defaults, enhanced with our custom font stack.

## Font Stack

*   **Primary Font**: System UI font stack (Segoe UI, Roboto, Helvetica, Arial, sans-serif) for maximum performance and native feel.
*   **Monospace**: SFMono-Regular, Menlo, Monaco, Consolas, "Liberation Mono", "Courier New", monospace.

## Headings

All HTML headings, `<h1>` through `<h6>`, are available.

*   `<h1>` - Page Titles (2.5rem)
*   `<h2>` - Section Titles (2rem)
*   `<h3>` - Subsection Titles (1.75rem)

## Body Text

*   Default `font-size`: `1rem` (16px).
*   Default `line-height`: 1.5.
*   Color: `$ez-text-color` (#333).

## Helpers

*   `.lead`: Make a paragraph stand out.
*   `.text-muted`: Muted text color.
*   `.text-primary`: Primary brand color text.
