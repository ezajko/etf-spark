# Colors

We define a semantic color system in `src/scss/base/_variables.scss`.

## Brand Colors

*   **Primary** (`$ez-primary-color`): `#0d6efd` (Blue). Used for links, buttons, and active states.
*   **Secondary** (`$ez-secondary-color`): `#6c757d` (Grey). Used for secondary actions and borders.

## UI Colors

*   **Success**: `#198754` - Validations, completion states.
*   **Warning**: `#ffc107` - Alerts, cautions.
*   **Danger**: `#dc3545` - Errors, destructive actions.
*   **Info**: `#0dcaf0` - Informational messages.

## Text Colors

*   **Body**: `$ez-text-color` (#212529)
*   **Muted**: `$ez-text-muted` (#6c757d)
*   **White**: `#ffffff`

## Backgrounds

*   `.bg-light`: Light grey background.
*   `.bg-dark`: Dark background (often footer).
*   `.bg-white`: White background.
