# Slim Navigation (Utility Bar)

The **Slim Navigation** (also known as the Utility Bar or Top Bar) is the thin dark strip at the very top of the page. It hosts secondary but essential links.

## Component

**`HeaderSlim.astro`**
*   **Location**: `src/site/components/navigation/HeaderSlim.astro`
*   **SCSS**: `src/scss/components/navigation/_site-nav-slim.scss`
*   **Class**: `.site-nav-slim`

## Content

*   **University Links**: Links to the main University site (UNSA).
*   **Service Links**: Quick access to student services (Zamger, C2, Mail).
*   **Language Selector**: Toggle between languages (BHS/EN).
*   **Search**: Global site search trigger.

## Styling

This component typically uses a dark background (`$site-footer-bg`) with white text to differentiate it from the main white header.
