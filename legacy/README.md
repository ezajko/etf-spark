# Legacy Project Files

Ovo su originalni fajlovi prije modernizacije projekta.  
Potpuno funkcionalan statički sajt.

## Struktura

- `index.html` - glavna stranica
- `css/` - stilovi
- `js/` - JavaScript fajlovi
- `images/` - slike i resursi
- `design/` - dizajn sistem dokumentacija

## Pokretanje

Za pokretanje legacy sajta, otvoriti `index.html` datoteku u web browser-u.

## Napomena

Ovi fajlovi su sačuvani kao backup prije implementacije modernog toolinga (Vite, Astro, SCSS, Podman).  
Novi projekat se nalazi u root direktoriju.
